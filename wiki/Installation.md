# Installation

## Requirements

- **Minecraft Server**: Paper, Spigot, or Folia (1.20.4+)
- **Java**: 17 or higher

## Optional Dependencies

- **Vault** - Required for economy features
- **Lands** - For land-based keep inventory settings
- **GriefPrevention** - For claim-based keep inventory settings

## Installation Steps

1. Download `DynamicKeepInv-X.X.X.jar` from [Releases](https://github.com/Alexteens24/DynamicKeepInv/releases)
2. Stop your server
3. Place the JAR file in your `plugins` folder
4. Start your server
5. The plugin will generate default config files in `plugins/DynamicKeepInv/`

## Files Generated

```
plugins/DynamicKeepInv/
├── config.yml      # Main configuration
└── messages.yml    # Messages (supports multiple languages)
```

## Verify Installation

Run `/dki status` to check if the plugin is working correctly.
