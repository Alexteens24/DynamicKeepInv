## Quick Start

1. Download the plugin JAR
2. Put it in your `plugins` folder
3. Restart the server
4. Edit `plugins/DynamicKeepInv/config.yml` if needed

## Default Behavior

- **Day time**: Keep inventory ON (players keep items when they die)
- **Night time**: Keep inventory OFF (players drop items when they die)
